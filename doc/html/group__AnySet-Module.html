 AnySet is an associative container that contains a set of unique objects of any constructible type. Search, insertion, and removal have average constant-time complexity. Internally, the elements are not sorted in any particular order, but organized into buckets. Which bucket an element is placed into depends entirely on the hash of its value. template<class T , class = std::enable_if_t<detail::has_hash_specialization_v<T>>>
std::size_t compute_hash ( const T &  value)
related

Generic hash function.

Users can customize the behavior of AnyHash by providing an overload of compute_hash() in the same namespace that their own classes are declared in. AnyHash makes an unqualified call to compute_hash() like so:

using te::compute_hash;
return compute_hash(value);

This allows ADL to find overloads of compute_hash() in the namespace that value's type is defined in.

te::compute_hash() otherwise uses te::Hash to compute value's hash code. If te::Hash has no specialization for value's type, then te::compute_hash() is removed from overload resolution via SFINAE.

value- object to compute the hash code for.
